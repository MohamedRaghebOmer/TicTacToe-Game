# XO Game

## Overview
This is a simple Tic-Tac-Toe (X and O) game built using C# and Windows Forms (.NET Framework). The program allows two players to play against each other on the same computer.

## Features
- Interactive 3x3 grid for the game.
- Alternating turns between X and O.
- Game detects winning conditions and draws.
- User-friendly interface with simple design.

## How to Run
1. Open the solution in Visual Studio.
2. Build the project.
3. Run the application. The main form (`Form1`) will launch automatically.

## Requirements
- Windows OS
- .NET Framework
- Visual Studio (Community Edition recommended)

## Notes
- This is a beginner-friendly project for learning C# Windows Forms applications.
- All logic is implemented in `Form1.cs`.
